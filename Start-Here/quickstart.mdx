## Prerequisites

Before you begin, ensure you have:
- Mixlar Control desktop application installed
- ESP32-S3 hardware controller connected via USB
- Windows PC connected to your local network

## Web Controller Setup

Follow these steps to enable remote access through your web browser.

### Step 1: Open Plugin Settings

1. Launch **Mixlar Control** on your PC
2. Click the **Settings** icon in the top-right corner
3. Select **Plugins** from the left-side menu
4. Find and click on **Web Controller**

### Step 2: Enable the Plugin

1. Toggle the **Enable** switch to turn on Web Controller
2. You'll see four configuration tabs appear:
   - Configure
   - Security
   - Remote Access
   - Network

### Step 3: Configure Port (Optional)

1. Click the **Configure** tab
2. Set your preferred port (default: `5000`)
3. Click **Save**

<Note>
  If port 5000 is already in use, choose a different port like 5001 or 8080.
</Note>

### Step 4: Set Up Security

1. Click the **Security** tab
2. Create a **username** for login access
3. Create a strong **password**
4. Click **Save**

<Warning>
  Never use simple passwords like "admin" or "password". Use a strong password with letters, numbers, and symbols.
</Warning>

### Step 5: Access Locally

You can now access the web interface:

**On the same PC:**
```
http://localhost:5000
```

**From other devices (same network):**
```
http://YOUR_PC_IP:5000
```

<Info>
  Find your PC's IP address: Open Command Prompt and type `ipconfig`. Look for "IPv4 Address".
</Info>

## Remote Access with Tailscale (Recommended)

For secure access from anywhere, use Tailscale. This provides HTTPS encryption and VPN security.

### Why Tailscale?

<CardGroup cols={3}>
  <Card title="Secure" icon="lock">
    HTTPS encryption with VPN tunnel
  </Card>
  <Card title="Easy" icon="sparkles">
    One-click setup with automatic configuration
  </Card>
  <Card title="Anywhere" icon="globe">
    Access from any location, any network
  </Card>
</CardGroup>

### Setup Steps

1. **Download Tailscale**
   - Visit [tailscale.com/download](https://tailscale.com/download)
   - Install on your PC and mobile devices
   - Create a free Tailscale account

2. **Enable in Mixlar Control**
   - Open Web Controller plugin settings
   - Click the **Remote Access** tab
   - Click **Generate HTTPS Certificates** button
   - Wait for certificate generation to complete
   - Toggle **Enable Tailscale** switch

3. **Get Your Secure Link**
   - Click the **Network** tab
   - Copy the Tailscale secure link (starts with `https://`)
   - Save this link for accessing from anywhere

4. **Access Remotely**
   - Open the secure link on any device with Tailscale installed
   - Login with your username and password
   - Control your mixer from anywhere!

<Tip>
  The Tailscale link works on any network - home WiFi, mobile data, coffee shop, anywhere!
</Tip>

## iOS App

Prefer a native mobile experience? Download the Mixlar Control app.

### Download

1. Open the **App Store** on your iPhone or iPad
2. Search for **"Mixlar Control"**
3. Tap **Get** to download
4. Open the app

### Connect

The app will automatically discover your Mixlar Control instance on the local network.

For remote access via Tailscale:
1. Install Tailscale on your iOS device
2. Log in with the same Tailscale account
3. Open Mixlar Control app
4. Enter your Tailscale secure link
5. Login with your credentials

## What You Need

- Mixlar Control desktop app installed
- ESP32-S3 controller connected via USB

## Setup Steps

### 1. Enable Web Controller

1. Open **Mixlar Control**
2. Click **Settings** â†’ **Plugins**
3. Click **Web Controller**
4. Toggle **Enable** to ON

### 2. Configure Settings

Click through the four tabs:

**Configure Tab**
- Set port (default: 5000)

**Security Tab**
- Create username and password

**Remote Access Tab**
- Click **Generate HTTPS Certificates**
- Toggle **Enable Tailscale** (for remote access)

**Network Tab**
- Copy your connection URL

### 3. Connect

**Same Network:**
```
http://localhost:5000
```

**From Phone (Same WiFi):**
```
http://YOUR_PC_IP:5000
```

**From Anywhere (Tailscale):**
```
https://your-tailscale-url:5000
```

## Remote Access (Recommended)

For secure access from anywhere:

1. Download [Tailscale](https://tailscale.com/download)
2. Install on PC and phone
3. Sign in with same account
4. Use the Tailscale URL from Network tab

## iOS App

Download **Mixlar Control** from the App Store for a native mobile experience.

# Sliders

The sliders on your Mixlar Mix allow you to control audio levels, system actions, MIDI outputs, smart home features, and more.  
Each slider can be assigned multiple inputs or actions to create a custom workflow that fits your setup.

---

## Adding a Slider Source

On the main Mixlar Control screen, each slider has a **+** button underneath it.

Clicking this **+** icon opens the assignment window where you can choose what the slider will control.

## Multi-Binding

Each slider can control **multiple sources at once**.

For example:

- Control Discord + YouTube volume together  
- Control your mic volume + mic noise suppression  
- Control all browser-based audio as one group  

Just select more than one item in the assignment window before saving.

---

## Custom Slider Colors

To help you recognize sliders quickly, each slider can have a **custom color**.

- By default, the color matches your theme  
- Click the color box on the main screen to choose any color you want  
- Many users color-code by category (e.g., purple = music, green = chat, red = mic)

This is especially helpful on wireless setups or in low-light environments.
